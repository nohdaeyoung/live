# 🛠️ 개발

> 담당: 개발팀 & 감성엔지니어 🤖

## 목적
제품 기능을 안정적으로 구현하고 자동화된 개발 파이프라인을 운영합니다.

## 핵심 태스크
- **프론트/백엔드 구현**: Next.js 15 (React 18) + Firebase 기반 풀스택 개발
- **자동 PR · 커밋 대행**: 감성엔지니어가 관리자 승인 기반으로 코드 변경·커밋·push 수행
- **CI/CD 파이프라인 관리**: GitHub Actions 워크플로우 구성 및 유지보수

## 기술 스택
| 영역 | 기술 |
|---|---|
| 프론트엔드 | Next.js 15, React 18, Tailwind CSS, Framer Motion |
| 백엔드 | Firebase (Firestore, Realtime DB, Hosting) |
| AI 파이프라인 | Python, LangGraph (324-os/) |
| CI/CD | GitHub Actions → Firebase Hosting 자동 배포 |
| 에이전트 | OpenClaw (비서공주 메인 에이전트) |
