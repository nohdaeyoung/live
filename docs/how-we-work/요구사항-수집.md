# 💡 아이디어 · 기획 (요구사항 수집)

> 담당: 비서공주 👸 & 탐정요정 🧚

## 목적
CEO(마스터 사미사)의 명령을 접수하고, 시장 조사와 기획을 통해 실행 가능한 1-Pager 기획서를 생성합니다.

## 핵심 태스크
- **명령 접수 & 판단**: 마스터의 요청을 분석하여 기획이 필요한지, 바로 실행할 수 있는지 판단
- **시장 조사 (한/영 웹 검색 5회+)**: Researcher 에이전트가 경쟁사·시장 현황·Pain Point 조사
- **1-Pager 기획서 자동 생성**: Planner 에이전트가 5단계(전략→설계→메트릭→성장→자가검증) 기획서 생성

## 트리거 조건

| 마스터 요청 | 판단 | 액션 |
|---|---|---|
| "기획해줘", "플래닝하자" | 즉시 가동 | Researcher → Planner 파이프라인 실행 |
| "XX 기능 만들자" (신규) | 확인 후 가동 | "기획부터 돌릴까요?" 확인 |
| "버튼 색 바꿔" (단순) | 바로 실행 | Planner 불필요 |
